> Compressed LibreOffice for AWS Lambda

## Install

```
$ yarn add aws-lambda-libreoffice
```

## Usage

```js
const awsLambdaLibreoffice = require('aws-lambda-libreoffice');

awsLambdaLibreoffice('unicorns');
//=> 'unicorns & rainbows'
```
